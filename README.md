Este projeto implementa, analisa e demonstra o algoritmo Shell Sort em Python, atendendo aos requisitos da avaliação de Estrutura de Dados.
Inclui:

Implementação limpa e comentada do Shell Sort

Ordenação crescente e decrescente

Testes com números, strings e objetos (Pessoa)

Medição de desempenho para diferentes tamanhos de entrada

Geração de gráficos comparativos

Módulo interativo de demonstração

SOBRE O PROJETO

O objetivo deste trabalho é investigar, na prática, o comportamento do algoritmo Shell Sort, comparando:

Tempo de execução em diferentes tipos de dados

Performance com listas pequenas e grandes

Diferença entre ordenação crescente e decrescente

Relação entre teoria da complexidade e resultados experimentais

O código foi implementado utilizando Python e estruturado de forma modular para facilitar testes e demonstrações.
